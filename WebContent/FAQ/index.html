  • Basics

  • How do I change/reset my password?

    To reset your password logout of the system and go to the login page. Don't login, instead click forgot password button and enter your email address. If the email address is on the system you should receive an email that allows you to reset your password. Once reset you can login with your new password.

  • How do I sign up?

    To register for the timetable program, a user must go to the login page and click on create an account. This will open up a form that allow students to register for the system. Once registered an email will be sent that you must confirm. Once that has been done return to login page and fill in your login details. ( Note: Lecturers must contact admin to be added to the system. )

  • How do I login ?

    To login into our system you must first be registered on it. Once registered you have to go to the login page and enter your email address on the username box and your password in the other box provided. Click login and once details are correct you can access the timetable management system.

  • Help once logged in

  • What help is available once logged in the program ?

    Once logged in you have a seperate help section that contains video's on how to use each of our features. If the user is still having trouble using the system we have also provided a contact form which allows users contact the administrator with concerns.

  • How do I contact the administrator ?

    If you are having any problems with using the system or want to report any error's or strange behaviour please contact the administrator. This can be done by using the contact form which has been included in the program and at the end of this page.
